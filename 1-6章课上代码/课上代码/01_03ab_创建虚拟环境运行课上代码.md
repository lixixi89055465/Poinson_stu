# 激活虚拟环境（Windows）
进入cmd命令行以后,然后把  虚拟环境文件夹下\Scripts\activate
![](/i/6812244b-fd15-4eec-b87c-7acb36405769.jpg)
激活虚拟
![](/i/0e0cea51-54a6-4bf0-9640-ce5cee01240f.jpg)
然后再pip install 包名,就可以在这个虚拟环境下安装包





把requirements.txt 所在目录cd进入到cmd命令行,然后执行命令:
pip install -r requirements.txt 
# 激活虚拟环境（macOS/Linux）
source 虚拟环境文件夹下/bin/activate

# 确认环境中有安装的包
pip list
